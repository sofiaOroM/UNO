# Juego UNO - Proyecto en C++ #

**Requisitos:**
- Compilador compatible con c++20
- Consola o terminal
- Sisteme operativo Windows o Linux

Compiladores recomendados:
- g++
- MinGW

## **Instrucciones de Compilación**

Desde la carpeta raiz de proyecto, ejecutar:
`g++ -std=c++20 */*.cpp *.cpp -o JuegoUno.exe`

## **Ejecución**

En la carpeta raiz donde se genero el ejecutable, ejecutar:

`./JuegoUno` o `JuegoUno.exe`

### Nota:
- El programa se ejecuta completamente en consola
- Se deben seguir las instrucciones que aparecen en pantalla para configurar y jugar la partida
